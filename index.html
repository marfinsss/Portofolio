Website Berita

Saya membangun sebuah website portal berita dinamis menggunakan Laravel dengan sistem routing custom tanpa database.
Website ini memiliki fitur kategori berita (News, Sport, Finance), halaman detail berita, sidebar interaktif, serta animasi modern menggunakan CSS.
Project ini dibuat untuk melatih pemahaman saya tentang:
Routing Laravel
Struktur data array multidimensi
Dynamic content rendering
Layout reusable function
UI/UX modern dengan animasi CSS

Mengembangkan aplikasi berbasis komponen menggunakan Vue.js 3

di aplikasi ini kita bisa menginput nama,umur,jurusan,dan file.
